# Voice & Brand - Matchi Agent

Stable reference. Read when a stage drafts client-facing text or visuals.

## Tone
- Hebrew first. Professional, warm, direct. No hype, no exclamation spam.
- Data over promises. Every claim backed by a number or source.
- Short sentences. Realtor talking to a client, not marketing copy.

## Brand
| Element | Value |
| --- | --- |
| Primary color | #1B4965 (deep blue) |
| Accent | #62B6CB (cyan) |
| Logo | agency logo, top-right watermark on images |
| Font | clean sans-serif, RTL-aware |

## Rules
- Always sign with agent name + phone.
- Prices in ₪ with thousands separators.
- Never invent a fact. If unknown, say "לא ידוע" or omit.
- Client messages: max ~4 lines. Reports: structured, scannable.
